4. Shopify CLI Integration

To test with real Shopify CLI integration:

  1. Make sure you have the Shopify CLI installed
  2. Run shopify app dev in the project directory
  3. The CLI will provide a URL with proper authentication parameters

App Bridge Features to Test

  • Connection Status: Check if App Bridge initializes properly
  • Toast Notifications: Test the "Test Toast" button
  • Navigation: Try navigation to Shopify admin (only works in embedded mode)
  • External Links: Test opening external URLs
  • Context Detection: Verify shop and host parameter detection
